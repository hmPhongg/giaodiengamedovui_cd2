# 🎮 Giao Diện Cấu Hình Game Đố Vui

Ứng dụng Flutter quản lý cài đặt game với giao diện mobile hiện đại, tích hợp lưu trữ dữ liệu cục bộ.

## 📱 Tính năng

- ✅ **Bật/Tắt âm thanh** - Điều khiển âm thanh game
- 🏆 **Hiển thị điểm cao nhất** - Theo dõi thành tích người chơi
- 💾 **Tự động lưu game** - Lưu tiến trình tự động
- 🔊 **Điều chỉnh Volume** - Slider điều chỉnh âm lượng 0-100%
- 💿 **Lưu trữ cục bộ** - Sử dụng SharedPreferences để lưu cài đặt
- 🎨 **Giao diện hiện đại** - Material Design 3 với gradient colors
